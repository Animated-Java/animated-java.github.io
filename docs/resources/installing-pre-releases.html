Installing Pre-Releases

Learn how to install pre-release versions of Animated Java.

  1. Step 1

    Open Blockbench, and navigate to File > Plugins > Load Plugin from File.

  2. Step 2

    Find the animated-java.js file you downloaded from our Discord Server, then click Open.

That's it! You should now have the pre-release version of Animated Java installed. If you have any issues, feel free to ask for help in the #animated-java-support channel of our Discord server.
