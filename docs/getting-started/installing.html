Installing Animated Java

What you’ll need

  • Blockbench (version 5.1.4 or higher)
  • Minecraft: Java Edition (version 1.20.4 or higher)

Basic Installation

  1. Open Blockbench and navigate to File > Plugins... to open the plugin manager.

  2. Search for “Animated Java” in the plugin marketplace.

  3. Click the “Install” button.

Advanced Installation (URL)

If you prefer to install the plugin directly from a URL, follow these steps:

  1. Open Blockbench and navigate to File > Plugins... to open the plugin manager.

  2. Click the “Install from URL” button.

  3. Enter the following URL in the input field:

    https://github.com/Animated-Java/animated-java/releases/latest/download/animated_java.js

Advanced Installation (File)

If you have downloaded the plugin as a .js file, you can install it manually:

  1. Open Blockbench and navigate to File > Plugins... to open the plugin manager.

  2. Click the “Install from File” button.

  3. Select the downloaded .js file from your computer and click “Open” to install the plugin.
